# NeuroWeb Studio Website

Sito web responsivo e multilingua per studio di sviluppo web.

## 🌐 Lingue
- Italiano
- Inglese  
- Tedesco

## 🚀 Funzionalità
- Design responsive
- Sistema multilingua JSON/JavaScript
- Carrello servizi
- Chatbot integrato
- Form contatti

## 📁 Struttura
- `index.html` - Homepage principale
- `servizi.html` - Pagina servizi dettagliata
- `css/` - Fogli di stile
- `js/` - Script JavaScript
- `lang/` - Traduzioni JSON
- `assets/` - Immagini e risorse
